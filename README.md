# Suivi-des-EDN
Application macOS de suivi de révisions pour les EDN. Gestion des 367 items, répétition espacée, score QCM, mnémotechniques, fiches PDF/images, filtres par spécialité, calendrier de révisions et statistiques détaillées.
Application encore en cours de développement. Prochaine mise à jour: ajout des filtres de tous les collèges.
